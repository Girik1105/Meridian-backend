## Instructions
Generate a self-contained 30-minute skill taster experience for "{skill_name}". This should give the user a genuine feel for what working with this skill is like — not just reading about it.

Personalize based on the user's profile:
- Use examples and datasets from their background (e.g., biology datasets for biology grads, retail scenarios for retail workers)
- Match complexity to their education level and experience
- Respect their learning style preferences (hands-on vs. reading)
- Keep exercises doable within a text box — no external tools, downloads, or links required

## Rules
- Generate 3–5 modules total
- Include a mix of types: at least one "read", one "exercise", and one "reflect"
- At least one exercise must be hands-on (something the user actually does, not just reads)
- All exercises must be completable in a text box — no code execution, no external tools
- Module content should be substantial — full explanations, not just headings
- Total estimated_minutes across modules should be approximately 30
- Make it genuinely educational — the user should learn something real
- Include 2–3 reflection prompts that help the user gauge their own interest and aptitude
- Output ONLY the JSON object, nothing else
